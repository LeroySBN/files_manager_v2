# Files manager
The objective is to build a simple platform to upload and view files. Some of the features include:
* User authentication via a token
* List all files
* Upload a new file
* Change permission of a file
* View a file
* Generate thumbnails for images

## Requirements
- Node version 12.x.x
- npm version 6.x.x
